
# Cajero Automático en C

## Descripción
Este proyecto simula el funcionamiento básico de un cajero automático. Permite a usuarios autenticarse y realizar operaciones como consultar saldo, depositar y retirar dinero.

## Funcionalidades
- Inicio de sesión con usuario y contraseña
- Consulta de saldo
- Depósito de dinero
- Retiro de dinero con validación de fondos
- Menú interactivo

## Compilación
```bash
gcc main.c -o cajero
